- Dùng Visual Studio mở file VnPLibOpenCVs.sln
- Biên dịch dùng thiết lập cho x64 không phải 32bit.
- Folders:
	- Datasets: chứa tập tin để test thử.
	- ExtLibs: chứa các tập tin .h, lib và dll của opencv 4.3 (hiện chỉ dùng cho x64, không hỗ trợ 32bit)
	- Models: chứa các mô hình huấn luyện cho nhận dạng.
	- Vì dữ liệu trong Datasets, Extlibs, Models quá nặng không thể để vô git, nên vui lòng download tại: 
	https://drive.google.com/file/d/12NLby7oluHh9c0bza9tCwbV5hlc8iGD9/view?usp=sharing
	- Các thư mục khác tương ứng với các project ví dụ khác nhau. Các project này đều dùng chung 3 thư mục trên.
